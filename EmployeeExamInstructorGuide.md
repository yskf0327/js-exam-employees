## 2. 実装のヒント・解説ポイント

### 年齢計算のロジック
多くの学生が単に `年 - 年` だけで済ませがちです。
`月` や `日` の比較を行い、**「今年の誕生日がまだ来ていないなら -1 する」** という条件分岐が書けているかを重点的に見てください。

```javascript
// 推奨実装例
let age = today.getFullYear() - birthDate.getFullYear();
const m = today.getMonth() - birthDate.getMonth();
if (m < 0 || (m === 0 && today.getDate() < birthDate.getDate())) {
    age--;
}
```

### 給与テーブルの参照
`switch` 文や `if` 文で分岐するのではなく、オブジェクト（連想配列）をマスターデータとして使い、`TABLE[key]` の形式で値を引く方法は、実務でも頻出のパターンです。このメリット（保守性、可読性）を解説してください。

### モーダル操作
BootstrapのモーダルJSを使わず、HTML5標準の `<dialog>` タグと `showModal()` / `close()` メソッドを使用しています。これにより外部ライブラリへの依存を減らし、JavaScript本来のDOM操作を学習させることができます。
